wallabag documentation
======================

.. image:: ../img/wallabag.png
   :alt: wallabag logo
   :align: center

**wallabag** É uma aplicação "leia mais tarde": Ele salva páginas da web mantendo apenas o conteúdo. Elementos como ícones de navegação ou propaganda são deletedos.
